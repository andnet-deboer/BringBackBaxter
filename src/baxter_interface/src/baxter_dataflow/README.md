
#### Signal

Follows an [observer pattern](https://www.geeksforgeeks.org/system-design/observer-pattern-set-1-introduction/). A signal is emitted by a source and received by one or more listeners. The signal class is responsible for managing the connections between sources and listeners, and for emitting signals to the appropriate listeners when an event occurs.
